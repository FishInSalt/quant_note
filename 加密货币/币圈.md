币圈术语

这是加密货币圈常见的专业术语，我来逐一解释：

## 你提到的术语

**发币**：指项目方创建并发行新的加密货币或代币。常见方式包括 ICO（首次代币发行）、IEO（交易所首发）、IDO（去中心化交易所首发）等。

**合约**：指加密货币衍生品交易，主要有永续合约和交割合约。允许使用杠杆做多或做空，风险较高，可能爆仓。

**U商**：专门从事 USDT（泰达币）等稳定币场外交易（OTC）的商人，负责法币与加密货币之间的兑换。

**返佣**：通过邀请链接注册后，邀请人可获得被邀请人交易手续费的一部分作为佣金返还。

**现货**：直接买卖加密货币本身，一手交钱一手交货，不涉及杠杆，相对风险较低。

## 其他常见圈内术语

| 术语 | 含义 |
|------|------|
| **梭哈** | 全部资金一次性投入 |
| **爆仓/清算** | 合约保证金不足被强制平仓 |
| **割韭菜** | 庄家或项目方收割散户 |
| **HODL** | 长期持有不卖（Hold的变体） |
| **FOMO** | 害怕错过行情而追涨 |
| **FUD** | 恐惧、不确定、怀疑，指负面消息 |
| **土狗/Meme币** | 无实际价值的投机代币 |
| **空投（Airdrop）** | 项目方免费发放代币 |
| **撸毛** | 通过交互获取空投奖励 |
| **Gas费** | 链上交易手续费 |
| **钱包地址** | 类似银行账号的加密货币地址 |
| **私钥/助记词** | 控制钱包资产的密码，泄露即丢币 |
| **链上/链下** | 在区块链上/不在区块链上 |
| **DeFi** | 去中心化金融 |
| **DEX/CEX** | 去中心化交易所/中心化交易所 |
| **挖矿** | 通过算力或质押获取代币 |
| **质押（Staking）** | 锁仓代币获取收益 |
| **白名单** | 有资格参与项目早期认购的名单 |
| **巨鲸** | 持有大量代币的大户 |
| **拉盘/砸盘** | 拉高/打压价格 |
| **插针** | 价格瞬间剧烈波动后恢复 |
