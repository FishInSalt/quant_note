## 维度一：按时间周期/交易频率

```
┌─────────────────────────────────────────────────────────┐
│                 按交易频率分类                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  类别        持仓时间      交易频率      典型策略        │
│  ────       ──────────   ──────────   ────────────     │
│                                                         │
│  超高频      毫秒-秒      数千次/天     做市、闪电套利   │
│  (HFT)      │                                          │
│             │ 特点：                                    │
│             │ - 需要极低延迟（<1ms）                    │
│             │ - 需要交易所co-location                  │
│             │ - 需要专业硬件和网络                      │
│             │ - 散户基本无法参与                        │
│             │                                          │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  高频        秒-分钟      数百次/天     剥头皮、微套利   │
│             │                                          │
│             │ 特点：                                    │
│             │ - 单笔利润很小（0.01-0.1%）               │
│             │ - 靠次数累积收益                          │
│             │ - 对手续费极其敏感                        │
│             │ - 需要API直连和快速执行                   │
│             │                                          │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  中频        分钟-小时    10-100次/天   日内趋势、网格   │
│  (日内)     │                                          │
│             │ 特点：                                    │
│             │ - 当日开仓当日平仓                        │
│             │ - 不持仓过夜，避免隔夜风险                │
│             │ - 对技术分析依赖较高                      │
│             │ - 适合中小资金                            │
│             │                                          │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  中低频      小时-天      1-10次/天     波段交易、突破   │
│  (波段)     │                                          │
│             │ 特点：                                    │
│             │ - 捕捉中等级别的价格波动                  │
│             │ - 持仓几小时到几天                        │
│             │ - 手续费影响较小                          │
│             │ - 适合兼职交易者                          │
│             │                                          │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  低频        天-周        几次/周       趋势跟踪、配置   │
│  (中线)     │                                          │
│             │ 特点：                                    │
│             │ - 捕捉较大的趋势性行情                    │
│             │ - 持仓几天到几周                          │
│             │ - 需要较大的止损空间                      │
│             │ - 对心理素质要求较高                      │
│             │                                          │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  超低频      周-月-年     几次/月       宏观配置、定投   │
│  (长线)     │                                          │
│             │ 特点：                                    │
│             │ - 基于基本面和宏观周期                    │
│             │ - 持仓几周到几年                          │
│             │ - 交易成本忽略不计                        │
│             │ - 更像投资而非交易                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**频率与资金规模的关系：**

```
┌─────────────────────────────────────────────────────────┐
│            交易频率 vs 适合资金规模                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  频率          适合资金        原因                      │
│  ────         ────────       ──────────────────        │
│                                                         │
│  超高频        >$1000万       基础设施成本高             │
│  高频          >$100万        需要规模效应覆盖成本       │
│  中频          $1万-$100万    手续费占比适中             │
│  中低频        $1000-$50万    灵活度高                  │
│  低频          任意规模        成本不敏感               │
│                                                         │
│  散户（<$10万）最适合：中频 到 低频 策略                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 维度二：按策略逻辑/原理

```
┌─────────────────────────────────────────────────────────┐
│                 按策略逻辑分类                           │
├─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  【趋势跟踪类】                                          │
│                                                         │
│  核心假设：价格会延续当前方向运动                        │
│  盈利来源：捕捉大趋势的主要部分                          │
│                                                         │
│  ├── 均线策略                                           │
│  │   └── 金叉死叉、多均线排列、MACD                     │
│  │                                                      │
│  ├── 通道突破                                           │
│  │   └── 唐奇安通道、布林带突破、ATR通道                │
│  │                                                      │
│  ├── 动量策略                                           │
│  │   └── 价格动量、相对强度、突破确认                   │
│  │                                                      │
│  └── 海龟交易法                                         │
│      └── 经典趋势跟踪系统                               │
│                                                         │
│  特点：胜率低（30-40%），盈亏比高（2:1以上）             │
│  适合：趋势明显的市场，如牛市/熊市                       │
│  不适合：震荡市                                          │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  【均值回归类】                                          │
│                                                         │
│  核心假设：价格会回归到某个均值水平                      │
│  盈利来源：价格过度偏离后的回归                          │
│                                                         │
│  ├── 超买超卖策略                                       │
│  │   └── RSI超买超卖、KDJ超买超卖                       │
│  │                                                      │
│  ├── 布林带回归                                         │
│  │   └── 触及上轨做空、触及下轨做多                     │
│  │                                                      │
│  ├── 统计套利                                           │
│  │   └── 配对交易、价差回归                             │
│  │                                                      │
│  └── 网格交易                                           │
│      └── 固定区间内高抛低吸                             │
│                                                         │
│  特点：胜率高（60-70%），盈亏比低（1:1左右）             │
│  适合：震荡市、区间波动                                  │
│  不适合：单边趋势市场（会持续亏损）                      │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  【套利类】                                              │
│                                                         │
│  核心假设：相关资产价格应该保持特定关系                  │
│  盈利来源：价格关系偏离后的收敛                          │
│                                                         │
│  ├── 跨交易所套利                                       │
│  │   └── 同一资产在不同交易所的价差                     │
│  │                                                      │
│  ├── 期现套利                                           │
│  │   └── 现货与期货/永续之间的基差                      │
│  │                                                      │
│  ├── 跨期套利                                           │
│  │   └── 不同到期日合约之间的价差                       │
│  │                                                      │
│  ├── 三角套利                                           │
│  │   └── A→B→C→A 汇率不一致                            │
│  │                                                      │
│  └── 资金费率套利                                       │
│      └── 永续合约资金费率收割                           │
│                                                         │
│  特点：理论上无风险，实际有执行风险                      │
│  适合：有技术能力搭建系统的交易者                        │
│  难点：机会稀少、竞争激烈、利润薄                        │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  【做市类】                                              │
│                                                         │
│  核心假设：通过提供流动性赚取买卖价差                    │
│  盈利来源：Bid-Ask Spread（买卖价差）                   │
│                                                         │
│  ├── 传统做市                                           │
│  │   └── 同时挂买单和卖单，赚取价差                     │
│  │                                                      │
│  ├── AMM流动性提供                                      │
│  │   └── 在DEX提供流动性，赚取手续费                    │
│  │                                                      │
│  └── 订单簿做市                                         │
│      └── 根据市场微观结构动态调整报价                   │
│                                                         │
│  特点：需要双边库存、面临库存风险                        │
│  适合：有充足资金和技术能力的机构                        │
│  风险：单边行情时库存贬值                                │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  【事件驱动类】                                          │
│                                                         │
│  核心假设：特定事件会引起可预测的价格变动                │
│  盈利来源：事件发生前后的价格变化                        │
│                                                         │
│  ├── 新闻/公告交易                                      │
│  │   └── 重大新闻发布后的快速反应                       │
│  │                                                      │
│  ├── 代币解锁交易                                       │
│  │   └── 在代币解锁前后进行交易                         │
│  │                                                      │
│  ├── 上币交易                                           │
│  │   └── 交易所上币公告后的交易                         │
│  │                                                      │
│  ├── 减半/升级交易                                      │
│  │   └── BTC减半、ETH升级等事件                         │
│  │                                                      │
│  └── 社交媒体驱动                                       │
│      └── KOL喊单、热点话题                              │
│                                                         │
│  特点：需要快速信息获取和执行                            │
│  适合：有信息优势或快速执行能力的交易者                  │
│  风险：信息可能已被Price In                             │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  【机器学习类】                                          │
│                                                         │
│  核心假设：历史数据中存在可学习的模式                    │
│  盈利来源：模型预测能力超过随机                          │
│                                                         │
│  ├── 监督学习                                           │
│  │   └── 分类（涨/跌）、回归（预测价格）                │
│  │                                                      │
│  ├── 强化学习                                           │
│  │   └── 通过交互学习最优交易策略                       │
│  │                                                      │
│  ├── 深度学习                                           │
│  │   └── LSTM、Transformer预测时序                     │
│  │                                                      │
│  └── NLP情感分析                                        │
│      └── 分析新闻、社交媒体情绪                         │
│                                                         │
│  特点：需要大量数据和计算资源                            │
│  适合：有ML背景的交易者                                  │
│  风险：过拟合、市场regime变化                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 维度三：按交易市场类型

```
┌─────────────────────────────────────────────────────────┐
│                 按市场类型分类                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【现货市场策略】                                        │
│  ├── 只能做多，不能做空                                 │
│  ├── 无杠杆或低杠杆（部分交易所提供）                   │
│  ├── 无爆仓风险                                         │
│  ├── 适合策略：趋势跟踪、定投、网格                     │
│  └── 资金效率较低                                       │
│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  【永续合约市场策略】                                    │
│  ├── 可做多可做空                                       │
│  ├── 高杠杆（1x-125x）                                  │
│  ├── 有爆仓风险                                         │
│  ├── 有资金费率机制                                     │
│  ├── 适合策略：双向趋势、对冲、资金费率套利             │
│  └── 资金效率高，但风险大                               │
│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  【交割合约市场策略】                                    │
│  ├── 有到期日                                           │
│  ├── 可做多可做空                                       │
│  ├── 有基差（期货vs现货价差）                           │
│  ├── 适合策略：期现套利、跨期套利、日历价差             │
│  └── 需要考虑展期                                       │
│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  【期权市场策略】                                        │
│  ├── 非线性收益结构                                     │
│  ├── 可构建复杂收益曲线                                 │
│  ├── 适合策略：波动率交易、Delta中性、期权套利          │
│  └── 需要理解Greeks（Delta/Gamma/Theta/Vega）          │
│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  【DeFi市场策略】                                        │
│  ├── 去中心化交易所（DEX）                              │
│  ├── 借贷协议                                           │
│  ├── 流动性挖矿                                         │
│  ├── 适合策略：DEX套利、清算机器人、MEV                 │
│  └── 需要理解智能合约和Gas费                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

│                                                         │

│  ├── 逐笔成交数据（Tick Data）                          │

│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │

│  └── 应用：市场情绪分析、极端位置识别                   │
│                                                         │
│  ──────────────────────────────────────────────────    │
│                                                         │
│  【情绪/社交数据】                                       │
│  ├── Twitter/X提及量和情绪                              │
│  ├── Reddit/Discord讨论热度                             │
│  ├── Google搜索趋势                                     │
│  ├── 恐惧贪婪指数                                       │
│  ├── 新闻情感分析                                       │
│  └── 应用：情绪驱动策略、反向指标                       │
